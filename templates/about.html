About page

This project is a very simple book recommendation system which, based on entered book name, recommends you similar books you might also like.

Recommendation is based on users who also liked your book.

Dataset used: http://www2.informatik.uni-freiburg.de/~cziegler/BX/

Tech stack: Python - Flask + WTForms, Heroku

Author: romanzdk.com, 2020

Github: https://github.com/romanzdk/books-recommender
